###SETUP THE DATABASE#####
	
Run the sql file to create the tables of grants

	mysql -u root -D fenix < createMWGrantTables.sql (etc\middleware\grantMigration)

Run the sql file to  load the created tables 
	
	mysql -u root -D fenix < loadMWGrantTables.sql (etc\middleware\grantMigration)

####CORRECT THE ORIGINAL DATABASE#####

Make the corrections of the mail received

	mysql -u root -D fenix < mailCorrectionsBD.sql (Docs\grant)
	
Corrrect teachers that refer the same person

	mysql -u root -D fenix < teacherDuplicatePersonCorrections.sql (Docs\grant)
	
Correct duplicated (document id number) persons

	mysql -u root -D fenix < correctDuplicateIDNumberEntriesInPessoa.sql (Docs\grant)
	
Delete teacher with no person associated

	mysql -u root -D fenix < deleteTeachersWithoutPerson.sql (Docs\grant)
	
Corrections in old tables for the adjustement of the new uniques
	
	mysql -u root -D fenix < correctUniquesInOldTables.sql (Docs\grant)
	
Corrections of repeated projects

	mysql -u root -D fenix < correctRepeatedProjects.sql (Docs\grant)
	-->VERIFY AFTER THIS SCRIPT IF THERE AREN'T REPEATED PROJCTES...
	
Corrections of bad entries in entidade pagadora

	mysql -u root -D fenix < correctDuplicatedEntriesInPaymentEntityOriginalTable.sql (Docs\grant)
